# crudSpring
A simple Groovy crud using Spring

## Features

- Create;
- Edit;
- Delete;
- Update;


### Requisites

  - Java 8+;
  - Gradle

### Installing

Clone or download files

Inside the Backend path run:

    gradle run
    

## Built With

  - Groovy 4.0.9
  - Spring 3.0.5
  - Gradle
